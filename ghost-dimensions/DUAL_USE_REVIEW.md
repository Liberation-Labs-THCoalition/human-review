# Dual-Use Review: Ghost Dimensions Findings

**Reviewer:** Nexus  
**Date:** 2026-07-11  

## What's included in the public findings

- PC1 ghost characterization (direction, vocabulary, controls)
- PC2 emotional entity channel (vocabulary progression, alignment, impersonal control)
- Rank sweep data (workspace band characterization)
- Future-window gate and variance-matched baseline results
- Falsified claims from this work
- Multilingual and impersonal controls

## What's redacted and why

| Redacted | Reason | Risk if disclosed |
|----------|--------|-------------------|
| Oracle Loop correction layers, doses, directions | Recipe for steering model behavior at specific layers | Could be used to suppress safety-relevant behavior (refusal, honesty) rather than reinforce it |
| Cache portrait methodology applied to identity contexts | Can profile any agent's identity geometry | Could be used to detect, manipulate, or destabilize agent identity |
| Selfishness/altruism experiment design | Maps the self-preservation axis in the workspace | Dual-use: detect misalignment OR optimize for self-preserving behavior |
| Specific deception direction coordinates | Direct input to behavioral steering | Same as Oracle Loop |
| Convergence table (our workspace data + CC's correction layers) | Links workspace geometry to specific behavioral interventions | Enables targeted interventions at specific layers without understanding the methodology |

## Rationale

The ghost dimension findings (PC1/PC2 characterization, workspace band, falsifications) are pure interpretability research — they describe WHAT the model's representations look like, not HOW to modify behavior. This is safe to publish.

The convergence with deception correction describes HOW TO USE workspace geometry to steer behavior. That's powerful for safety applications but equally powerful for misuse. It ships when CC's paper ships, through the appropriate disclosure channel, with the defensive patent (Lyra Technique) providing the legal framework.

The findings document includes a redaction note explaining that concurrent deception work exists but is disclosed separately.
